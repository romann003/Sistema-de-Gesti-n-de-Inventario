  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  ## Tests

  Unit and integration tests are configured with Vitest and Testing Library.

  - Install dev dependencies: `npm i` (already required for the project)
  - Run tests (interactive):

  ```
  npm run test
  ```

  - Run tests once (CI):

  ```
  npm run test:run
  ```

  - Run TypeScript type checks:

  ```
  npm run typecheck
  ```

  ## Thunder Client

  A sample Thunder Client collection is included at `thunder-requests/collection.json` with example requests. Import it into the Thunder Client extension and set the `baseUrl` variable (default `http://localhost:5173`).
